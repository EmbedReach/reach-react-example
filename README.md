<h2 align="center">🏃‍♂️ Getting Started</h2>

```bash
npm install
npm run dev
```

That's it! Visit `localhost:3000` to see the integration in action.

- 🎯 Implement the Segment Builder component
- 🎨 Customize the theme to match your brand
